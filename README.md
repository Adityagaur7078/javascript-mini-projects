## 📖 About

This repository contains **30 JavaScript Mini Projects** that I built while learning modern web development.

Each project focuses on improving JavaScript fundamentals, DOM manipulation, API integration, event handling, local storage, and problem-solving skills.

The primary goal of this repository is to practice building real-world applications before moving to React and Full-Stack Development.
